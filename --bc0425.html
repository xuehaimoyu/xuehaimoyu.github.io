Qiskit介绍（简单展示一下）

Qiskit是一个开源的量子计算软件开发框架，用于构建和运行量子计算程序。它提供了丰富的工具和库，使用户能够编写、部署和执行量子算法。Qiskit支持多种编程语言，如Python和C++，并且与多种量子计算平台兼容。

使用Qiskit，可以创建量子电路、定义量子门操作、模拟量子计算和进行量子计算实验。它还提供了许多实用工具和算法，以帮助用户解决各种问题，包括优化、化学计算和机器学习等领域。

Example1: qiskit示例程序 Bell State制备

制备一个Bell State

from qiskit import QuantumCircuit, transpile
from qiskit_aer import AerSimulator
from qiskit.visualization import plot_histogram
# Use Aer's AerSimulator
simulator = AerSimulator()
# Create a Quantum Circuit acting on the q register
circuit = QuantumCircuit(2, 2)
# Add a H gate on qubit 0
circuit.h(0)
# Add a CX (CNOT) gate on control qubit 0 and target qubit 1
circuit.cx(0, 1)
# Map the quantum measurement to the classical bits
circuit.measure([0, 1], [0, 1])
# Compile the circuit for the support instruction set (basis_gates)
# and topology (coupling_map) of the backend
compiled_circuit = transpile(circuit, simulator)
# Execute the circuit on the aer simulator
job = simulator.run(compiled_circuit, shots=1000)
# Grab results from the job
result = job.result()
# Returns counts
counts = result.get_counts(compiled_circuit)
print(counts)
print("\nTotal count for 00 and 11 are:", counts)
# Draw the circuit
circuit.draw("mpl")

可以根据测量结果看出确实是Bell State

2 Introduction to VQA

VARIATIONAL QUANTUM ALGORITHMS

参考文献：https://arxiv.org/pdf/2310.12571.pdf

以目前的实验手段能够制造出来的量子计算机一般称为带噪声的中型量子计算机（NISQ）

拥有 101 10210^1 ~ 10^2101 102 量级的量子比特，并且无法排除噪声的影响，量子线路的宽度和深度都受到严重的限制。

这导致一些著名的教科书上的量子算法并不能在近期获得实现。

如 量子相位估计算法，以此为基础的 Shor算法、HHL 算法；以及 Grover 算法 等。

开发适用于 NISQ 时代的量子算法实现 quantum advantage，是目前的热点。

2.2 QAOA Quantum approximate optimization algorithm

量子近似优化算法

初始文献：[1411.4028] A Quantum Approximate Optimization Algorithm (arxiv.org)

2.2.1 Brief introduction of QAOA

于2014年提出，是一种全新的量子-经典混合算法，可以求解组合优化问题的近似解，被广泛认为有潜力在中等嘈杂型量子(NISQ)计算芯片上展现量子优势。近年来也有很多的工作围绕这种算法展开，2021年谷歌使用“Google Sycamore”量子计算机，真正对非硬件原生问题和硬件原生问题展开研究，探究了在硬件系统上编译和未编译QAOA问题的表现，北京量子院也开发了图分解的算法用以真正将这种算法应用于NISQ上。

QAOA也可以广泛应用于除了物理之外的其他领域，如金融等。

参考文献：Quantum approximate optimization of non-planar graph problems on a planar superconducting processor | Nature Physics

表明可能具有量子优势及可尝试性的论文：

[1602.07674] Quantum Supremacy through the Quantum Approximate Optimization Algorithm (arxiv.org)

对于给定的NP-Hard问题，近似算法是一种多项式时间算法，该算法以期望的一些质量保证来解决每个问题实例。品质因数是多项式时间解的质量与真实解的质量之间的比率。

组合优化问题

组合优化问题广泛应用于科研和生活中的各个领域，包括路径规划，网络设计，物理问题的仿真计算等。这是一类在离散状态下求极值的最优化问题，直观上可以通过穷举法解决问题，但是解的数量会随着问题的规模而指数增大，因此目前发展了很多算法来解决此类问题。这些算法可以分为精确算法和近似优化算法。精确算法如分支定界法可以给出全局下最优，但依旧会随着问题规模的扩大而消耗巨大的计算资源。近似优化算法可以找出组合优化问题局部最优，对计算资源消耗较小。量子近似优化算法正是一类经典-量子混合的近似方法，将指数级时间降为多项式级。

import numpy as np
from scipy.linalg import block_diag
from scipy.optimize import minimize

np.set_printoptions(precision=4,suppress=True)#控制小数精度


# Pauli matrices
I  = np.array([[ 1, 0],
               [ 0, 1]])
Sx = np.array([[ 0, 1],
               [ 1, 0]])
Sy = np.array([[ 0,-1j],
               [1j, 0]])
Sz = np.array([[ 1, 0],
               [ 0,-1]])

# Hadamard matrix
H = (1/np.sqrt(2))*np.array([[ 1, 1],
                             [ 1,-1]])

# Phase matrix
S = np.array([[ 1, 0],
              [ 0,1j]])

# single qubit basis states |0> and |1>
q0 = np.array([[1],
               [0]])
q1 = np.array([[0],
               [1]])

# Projection matrices |0><0| and |1><1|
P0  = np.dot(q0,q0.conj().T)
P1  = np.dot(q1,q1.conj().T)


# Rotation matrices as a function of theta, e.g. Rx(theta), etc.
Rx = lambda theta : np.array([[    np.cos(theta/2),-1j*np.sin(theta/2)],
                              [-1j*np.sin(theta/2),    np.cos(theta/2)]])
Ry = lambda theta : np.array([[    np.cos(theta/2),   -np.sin(theta/2)],
                              [    np.sin(theta/2),    np.cos(theta/2)]])
Rz = lambda theta : np.array([[np.exp(-1j*theta/2),                0.0],
                              [                0.0, np.exp(1j*theta/2)]])

# CNOTij, where i is control qubit and j is target qubit
CNOT10 = np.kron(P0,I) + np.kron(P1,Sx) # control -> q1, target -> q0
CNOT01 = np.kron(I,P0) + np.kron(Sx,P1) # control -> q0, target -> q1

SWAP   = block_diag(1,Sx,1)

g0 = -1.052373245772859
g1 = 0.39793742484318045
g2 = -0.39793742484318045
g3 = -0.01128010425623538
g4 = 0.18093119978423156

#哈密顿量在这里
Hmol = (g0 * np.kron( I, I) + # g0 * I
        g1 * np.kron( I,Sz) + # g1 * Z0
        g2 * np.kron(Sz, I) + # g2 * Z1
        g3 * np.kron(Sz,Sz) + # g3 * Z0Z1
        g4 * np.kron(Sx,Sx))  # g5 * X0X1

electronic_energy = np.linalg.eigvalsh(Hmol)[0] # take the lowest value
print("Classical diagonalization: {:+2.8} Eh".format(electronic_energy))

# initial basis, put in |01> state with Sx operator on q0
psi0 = np.zeros((4,1))
psi0[0] = 1
psi0 = np.dot(np.kron(I,Sx),psi0)
print(psi0)

# read right-to-left (bottom-to-top?)
ansatz = lambda theta: (np.dot(np.dot(np.kron(-Ry(np.pi/2),Rx(np.pi/2)),
                        np.dot(CNOT10, 
                        np.dot(np.kron(I,Rz(theta)),
                               CNOT10))),
                               np.kron(Ry(np.pi/2),-Rx(np.pi/2))))

result = minimize(projective_expected,theta,args=(ansatz,psi0))
